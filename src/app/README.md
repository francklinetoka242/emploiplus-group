# src/app

## Rôle

Ce dossier contient les couches d’infrastructure applicative de haut niveau : routage, providers, layouts et points d’entrée des routes.

## Ce qui doit y être placé

- configuration du router React Router
- providers globaux
- layouts applicatifs
- définitions de routes
- shells d’application

## Ce qui ne doit jamais y être placé

- logique métier spécifique à une feature
- composants UI génériques
- appels Supabase directs
- logique de domaine candidate/jobs/blog
