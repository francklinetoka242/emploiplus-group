# src/shared

## Rôle

Ce dossier contient tout ce qui est réutilisable au sein de l’application mais qui n’appartient à aucun domaine métier précis.

## Ce qui doit y être placé

- composants UI génériques
- primitives UI
- hooks partagés techniques
- utilitaires réutilisables

## Ce qui ne doit jamais y être placé

- logique métier liée aux offres, candidats, notifications ou blog
- appels API spécifiques à une feature
- dépendances fortes à un domaine particulier
