# src/features/candidates

## Rôle

Cette feature regroupe la logique métier liée aux candidats, à leur profil, à leurs données personnelles et à leurs parcours.

## Autorisé

- composants liés au candidat
- hooks de profil candidat
- API liées au candidat
- types de profil, expérience, compétences, langues, préférences

## Interdit

- composants UI génériques
- logique d’authentification externe
- accès direct à la logique jobs ou notifications
